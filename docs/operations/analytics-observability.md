# Analytics And Observability

## Product Analytics

- onboarding completion
- first plan generated
- task completion rate
- escalation conversion
- recovery action usage
- demo flow completion

## Technical Observability

- app errors
- webhook failures
- delayed job retries
- provider delivery failures
- escalation state mismatches

## Recommended Tools

- PostHog for product analytics
- Sentry for application errors
- structured logs for worker and webhook tracing
