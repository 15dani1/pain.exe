# Coach Profile

## Purpose

Separate persona intensity from product safety and consent rules.

## Fields

- display name
- tone descriptor
- profanity ceiling
- cadence
- escalation aggressiveness
- voice preset
- fallback-safe copy profile

## Rules

- Persona cannot override quiet hours, consent, or channel restrictions.
- The system should be able to swap from "Goggins-mode" to "inspired-by" without changing workflow code.
