# Demo Slice

## Hackathon Goal

Ship one polished end-to-end scenario that makes the product feel real.

## Seeded Demo Story

1. Demo user already has a goal and generated plan
2. A run is overdue and calendar timing has elapsed
3. The user has ignored an in-app reminder and SMS
4. The dashboard shows growing debt and a looming escalation
5. A voice call fires and the plan restructures into a punishment or redemption workout

## Build Order

1. Seeded data and static scenario
2. Daily command center UI
3. Escalation state machine
4. Message timeline and call summary
5. Calendar sync stub and one wearable verification path

## Out Of Scope For Initial Commit

- billing
- app-store packaging
- multi-provider wearable support
- admin console
- deep compliance workflows
