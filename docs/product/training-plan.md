# Training Plan

## Inputs

- goal
- deadline
- baseline fitness
- weekly schedule
- equipment
- injury constraints
- preferred workout windows

## Outputs

- dated task list
- primary and backup workout
- completion criteria
- milestone checkpoints
- missed-day recovery logic

## v1 Approach

- Generate a 2-6 week plan horizon for the demo
- Store daily tasks individually for easier escalation and calendar sync
- Regenerate on major disruptions while preserving old plan versions
