# Wearables Integration

## Goal

Accept one credible workout verification path in the hackathon demo.

## v1 Strategy

- Start with a single aggregator or one direct provider
- Ingest completed workout summaries and movement signals
- Match imported evidence against planned tasks
- Prevent redundant nagging when effort is already verified

## Matching Rules

- compare time window
- compare workout type
- compare duration or minimum effort threshold
